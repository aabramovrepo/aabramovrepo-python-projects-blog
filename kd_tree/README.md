# kd_tree
[Kd-tree and Nearest neighbor (NN) search (2D case)](https://salzi.blog/2014/06/28/kd-tree-and-nearest-neighbor-nn-search-2d-case/)

### Requirements

- Ubuntu Linux 20.04
- Python version >= 3.9

### Installation

```sh
>>> pip install -U -r requirements.txt
```

### How to Run

```sh
>>> python kd_tree.py <OUTPUT_DIR>
```
